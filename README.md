# plugin-3dBoxScan
## Compiling

* Delete old files if exists:

```bash
rm -Rf CMakeFiles
rm CMakeCache.txt
rm cmake_install.cmake
rm Makefile
rm pclBox
```

* Generate Makefile:

```bash
cmake .
```

* Compiling:

```bash
make
```

## Running

* Running:

```bash
./pclBox
````

## Operate:
* Clean desk;
* Press "b" and wait until program save background (about 20 seconds);
* Put the box on desk.
