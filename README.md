1. Compiling

    1.1. Delete old files if exists:
        rm -Rf CMakeFiles
        rm CMakeCache.txt
        rm cmake_install.cmake
        rm Makefile
        rm pclBox

    1.2. Generate Makefile:
        cmake .

    1.3. Compiling:
        make

2. Running

    2.1. Running:
        ./pclBox

    2.2. Operate:
        Clean desk;
        Press "b" and wait until program save background (about 20 seconds);
        Put the box on desk.
